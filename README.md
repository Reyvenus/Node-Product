#Notas
```
               //App de consola interactiva realizada en Node
 
    El funcionamiento de la aplicacion es confeccionar tablas de multiplicar con 
posterior guardado automatico en directorio definido.

 Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     La base muestra cual es la tabla de multiplicar
                                                            [número] [requerido]
  -l, --listar   Listar nos muestra la tabla de multiplicar y crea un archivo en
                 el directorio                       [booleano] [defecto: false]
  -h, --hasta    Muestra hasta que numero se tiene q multiplicar la base
                                              [número] [requerido] [defecto: 10]
```
